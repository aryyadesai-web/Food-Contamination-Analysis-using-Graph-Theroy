\# Food Supply Chain Contamination Analysis using Graph Theory



\## Description

A Graph Theory project that models a food supply chain network and analyzes contamination flow using NetworkX and Matplotlib.



\## Features

\- Directed and Undirected Graph Modeling

\- Shortest Path Analysis

\- Cycle Detection

\- Eulerian Path and Circuit Analysis

\- Connected Components

\- Minimum Spanning Tree (MST)

\- Articulation Points and Bridges

\- Planarity Testing

\- Graph Coloring



\## Supply Chain Stages

\- Farm

\- Collection

\- Processing

\- Warehouse

\- Distributor

\- Supermarket

\- Restaurant

\- Consumer



\## Technologies Used

\- Python

\- NetworkX

\- Matplotlib



\## How to Run



```bash

pip install networkx matplotlib

python main.py

```
